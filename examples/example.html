~headerText 
      This package is written in pure JS and has no dependencies. The ~packageName.js
      file can be included as a script element in any html webpage or static html file,
      and it will run templating functions on the contents of that page.
    

    Basics
        ~: Any variable name preceded by "~" in html
                  will be replaced with the value of the variable when rendered

        @: Any variable name preceded by "@" in html will be replaced with the value of the variable when rendered,
                  AND will dynamically udate if the variable changes

        for: For loops are created using the <for> tag with attributes of var, range, or list

        if: If statements are created using the <if> tag with attribute of condition

        switch: switch statements are created using the <switch> and <case> tags

        eval: strings can be evaluated in js using x

        escapes: strings can be escaped in html using <eval>xExamples
